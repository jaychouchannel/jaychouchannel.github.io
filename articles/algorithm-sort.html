深入理解 JavaScript 排序算法：从原理到实现

Jaychouchannel · 2023-06-15

本文介绍常见排序算法（冒泡、选择、插入、归并、快速、堆排序等）的核心思想、时间与空间复杂度，并提供 JavaScript 示例以便在实际项目中[...]

为什么排序重要

排序是计算机科学中的基础问题，合理选择排序算法可以在不同数据规模和数据分布下带来巨大性能差异。

常见排序算法简介

冒泡排序（Bubble Sort）

最简单的比较排序，稳定但时间复杂度一般为 O(n^2)。

// 冒泡排序示例（JavaScript）
function bubbleSort(arr) {
  const a = arr.slice();
  for (let i = 0; i < a.length; i++) {
    for (let j = 0; j < a.length - 1 - i; j++) {
      if (a[j] > a[j + 1]) {
        [a[j], a[j + 1]] = [a[j + 1], a[j]];
      }
    }
  }
  return a;
}

快速排序（Quick Sort）

通常性能非常好，平均时间复杂度 O(n log n)。

// 快速排序（递归版）
function quickSort(arr) {
  if (arr.length <= 1) return arr.slice();
  const a = arr.slice();
  const pivot = a[Math.floor(a.length / 2)];
  const left = [], right = [], equal = [];
  a.forEach(v => {
    if (v < pivot) left.push(v);
    else if (v > pivot) right.push(v);
    else equal.push(v);
  });
  return quickSort(left).concat(equal, quickSort(right));
}

归并排序（Merge Sort）

稳定，适合链表或需要稳定排序时使用，空间复杂度 O(n)。

复杂度对比与选择建议

  • 小规模数据（n < 50）：插入或简单的 O(n^2) 算法通常更快。
  • 通用场景：快速排序或归并排序。
  • 需要稳定排序且内存允许：使用归并排序。
