深入理解 JavaScript 排序算法：从原理到实现

Jaychouchannel
2023-06-15 · 8 分钟 阅读
算法 JavaScript 复杂度分析

排序是计算机科学中最经典的问题之一。本文会按稳定性、时间复杂度和典型实现来介绍常见排序算法，并给出 JavaScript 示例实现，帮助读者理解每种算法的适用场景。

冒泡排序（Bubble Sort）

原理：重复遍历数组，比较相邻元素并交换，最大的元素逐步“冒泡”到末尾。时间复杂度 O(n^2)，适合教学与小规模数据。

// 简单版冒泡排序
function bubbleSort(arr) {
  const a = arr.slice();
  for (let i = 0; i < a.length - 1; i++) {
    for (let j = 0; j < a.length - 1 - i; j++) {
      if (a[j] > a[j+1]) {
        [a[j], a[j+1]] = [a[j+1], a[j]];
      }
    }
  }
  return a;
}

快速排序（Quick Sort）

原理：分治法，选择基准元素将数组分为两部分，递归排序。平均时间 O(n log n)，但最坏 O(n^2)。适合通用场景。

归并排序（Merge Sort）

原理：稳定的分治法，时间复杂度始终 O(n log n)，空间复杂度 O(n)。在需要稳定排序或对大数据进行外部排序时非常有用。

实战建议

  • 一般情况下使用内置的 sort（配合合理的比较函数）或经过优化的快速/归并实现。
  • 当对稳定性有要求时优先考虑归并排序。
  • 大量数据时注意内存占用与稳定性权衡。
